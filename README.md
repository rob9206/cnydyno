## Deployment

This site is ready to deploy to Netlify. Choose your preferred method:

1. **GitHub Integration** - Connect repository for automatic deployments
2. **Netlify CLI** - Deploy from command line
3. **Drag & Drop** - Upload files directly

See [DEPLOYMENT.md](DEPLOYMENT.md) for complete instructions.
